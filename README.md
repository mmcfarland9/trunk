# Trunk

**Reap what you sow.** A personal growth tracker built around gardening metaphors.

```
web/     Vite + TypeScript (vanilla DOM, no framework)
ios/     Swift 5.9+ / SwiftUI (iOS 17+)
shared/  Constants, schemas, formulas
```

## Quick Start

```bash
cd web && npm install && npm run dev    # http://localhost:5173
cd ios && open Trunk.xcodeproj          # Build with Cmd+R
```
